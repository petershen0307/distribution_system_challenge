## Prepare Maelstrom

### install open jdk with asdf

```bash
asdf plugin-add java https://github.com/halcyon/asdf-java.git
asdf install java openjdk-21
asdf global java openjdk-21
```

### install graphviz gnuplot

```bash
sudo apt install graphviz gnuplot
```
